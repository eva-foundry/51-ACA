# 5. Public Sector Strategy

Your government contacts are a **very different environment**.

You already know this, but it’s important to be careful about:

* procurement rules
* conflict of interest
* departmental approvals

Instead of “selling”, the better approach is:

```
demonstration / knowledge sharing
```

Example positioning:

```
“I built a tool that analyzes Azure subscriptions and identifies cost optimization opportunities.”
```

Let them ask questions.

If they are interested:

```
pilot
proof of concept
```

Government adoption almost always starts that way.
